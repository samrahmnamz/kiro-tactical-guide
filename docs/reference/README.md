# Reference

This directory contains reference documentation for the Kiro Cloud Engineering/DevOps Toolbox.

## Purpose

Reference documentation provides detailed specifications and technical details:
- Hook configuration schema and options
- Spec document format and sections
- Steering rule syntax and capabilities
- MCP integration configurations
- API references for generated code
- Command-line tool references

## Contents

Reference materials serve as the authoritative source for configuration details, supported options, and technical specifications for all toolkit components.
